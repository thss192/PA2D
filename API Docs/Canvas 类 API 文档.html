📐 基础图形绘制

此模块提供了一系列用于在画布上绘制基础几何图形的函数。

💡 核心机制说明

  1. 🎨 颜色格式 (Color Format):

    • 所有涉及颜色的参数（如 fillColor, strokeColor）均接受 Color 类型。
    • Color 可由 uint32_t 隐式构造，格式严格遵循 0xAARRGGBB (Alpha, Red, Green, Blue)。
    • 示例: 0xFFFF0000 为不透明红色，0x8000FF00 为半透明绿色。

  2. 🖌️ 填充与描边 (Fill & Stroke):

    • 所有闭合图形（矩形、圆形、多边形等）均同时支持填充和描边。
    • 禁用填充: 将 fillColor 设置为 0 (即 pa2d::None 或全透明)。
    • 禁用描边: 将 strokeColor 设置为 0，或者将 strokeWidth 设置为 0。
    • 透明度叠加: opacity 参数 (0.0 - 1.0) 会与颜色的 Alpha 通道相乘，控制整体透明度。

📋 函数接口列表

图形类型 方法签名 详细功能说明
直线 line(float x0, float y0, float x1, float y1, const Color& color, float lineWidth) 绘制从 (x0, y0) 到 (x1, y1) 的直线。注意: 直线没有填充属性，只有颜色和线宽。
多段线 polyline(const std::vector<Point>& points, const Color& color, float lineWidth, bool closed) 连接一组点。如果 closed 为 true，则首尾相连。注意: 此函数仅绘制线条轮廓，不进行填充。
多边形 polygon(const std::vector<Point>& vertices, const Color& fillColor, const Color& strokeColor, float strokeWidth, float opacity) 绘制闭合多边形。若 fillColor 为 0 则不填充；若 strokeColor 为 0 则不描边。
三角形 triangle(float ax, float ay, float bx, float by, float cx, float cy, const Color& fillColor, const Color& strokeColor, float strokeWidth, float opacity) 绘制由三个顶点组成的三角形，支持填充和描边。
矩形 rect(float x, float y, float width, float height, const Color& fillColor, const Color& strokeColor, float strokeWidth, float opacity) 绘制标准矩形，(x, y) 为左上角坐标。
旋转矩形 rect(float centerX, float centerY, float width, float height, float angle, const Color& fillColor, const Color& strokeColor, float strokeWidth, float opacity) 绘制旋转矩形。注意: 坐标变为 中心点 (centerX, centerY)，angle 为旋转角度/弧度。
圆角矩形 roundRect(float x, float y, float width, float height, const Color& fillColor, const Color& strokeColor, float cornerRadius, float strokeWidth, float opacity) 绘制带圆角的矩形，cornerRadius 控制圆角的半径。
旋转圆角矩形 roundRect(float centerX, float centerY, float width, float height, float angle, const Color& fillColor, const Color& strokeColor, float cornerRadius, float strokeWidth, float opacity) 绘制旋转的圆角矩形，基于 中心点 坐标。
圆形 circle(float centerX, float centerY, float radius, const Color& fillColor, const Color& strokeColor, float strokeWidth, float opacity) 绘制圆形。
椭圆 ellipse(float cx, float cy, float width, float height, const Color& fillColor, const Color& strokeColor, float strokeWidth, float opacity) 绘制轴对齐椭圆， width/height 为外接矩形的宽高。
旋转椭圆 ellipse(float cx, float cy, float width, float height, float angle, const Color& fillColor, const Color& strokeColor, float strokeWidth, float opacity) 绘制旋转椭圆。
扇形 sector(float cx, float cy, float radius, float startAngle, float endAngle, const Color& fillColor, const Color& strokeColor, float strokeWidth, float opacity, bool drawArc, bool drawRadialEdges) 绘制扇形区域。drawArc: 是否绘制外圆弧描边。drawRadialEdges: 是否绘制两条半径描边。如果要绘制完整的饼图块，两者都设为 true。
